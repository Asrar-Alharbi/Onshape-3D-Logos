# 3D Olympic Rings Logo Design (Onshape) 🏅

A clean, precise, and fully-dimensioned 3D model of the **Olympic Rings Logo** designed using **Onshape** CAD software. This project demonstrates basic-to-intermediate CAD modeling techniques, including parametric sketching, linear patterns, and multi-part extrusion.

## 🛠️ Features

* **Parametric Design:** Built using strict geometric constraints and dimensions to ensure perfect symmetry.
* **Multi-Part Partitioning:** Optimized to export each ring as an independent body for easy multi-color 3D printing or rendering.
* **3D Printing Ready:** Includes a high-resolution, verified `.STL` file optimized with a flat base for direct slicing.

---

## 📐 Design Specifications

* **Software Used:** Onshape CAD
* **Units:** Millimeters ($mm$)
* **Design File Format:** `.STL` (Binary, High-Resolution)
* **Key Dimensions:**
  * **Outer Ring Diameter:** $50\text{ mm}$
  * **Inner Ring Diameter:** $42\text{ mm}$
  * **Center-to-Center Spacing:** $46\text{ mm}$
  * **Extrusion Thickness (Depth):** $2\text{ mm}$
